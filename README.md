# gnuplot-svg
Examples of gnuplot script to produce svg image


## execute
Use [gnuplot](https://github.com/gnuplot/gnuplot) in command line as follow

```
gnuplot a_script.gnu
```
where `a_script.gnu` is one of the provided examples.
